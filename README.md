# Fil d'actualité - Mately

Ceci est une application mobile réalisée en React Native qui permet de lire les posts utilisateurs d'un fil d'actualité, de créer des posts et également de les supprimer. Je l'ai développé à la suite de mon entretien avec l'entreprise Mately durant lequel nous avons évoqué un test pour jauger mes compétences.
